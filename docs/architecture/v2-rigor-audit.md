This audit classifies strictness by product value, not by age. Decisions are
initial recommendations for Checkpoint 1 review.

## Summary

The highest-value keep set is graph closure, trace authority, connector safety,
run lifecycle, path safety, report validation, catalog ownership, and generated
surface drift.

The highest-value simplification set is runner decomposition, authoring step
shape, route alias visibility, stage policy ownership, checkpoint policy
ownership, fanout decomposition, and projection boundaries.

The strongest demotion candidates are runtime-proof as architecture evidence,
method labels without consumers, and behavioral notes that have not yet become
tests.
